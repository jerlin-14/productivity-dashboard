# Personal Productivity Dashboard

## Overview

A responsive productivity dashboard built using HTML5, CSS3, and Vanilla JavaScript.

The application helps users manage daily tasks, track productivity, monitor time, and customize the interface using a persistent theme system.

## Features

### Live Clock
- Displays current system time
- Updates every second using JavaScript intervals

### Task Manager
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks by status:
  - All
  - Pending
  - Completed

### Task Statistics
- Total Tasks
- Completed Tasks
- Pending Tasks

### Persistent Dark/Light Mode
- Toggle between dark and light themes
- Theme preference stored using localStorage

### Stopwatch
- Start timer
- Stop timer
- Reset timer

### Responsive Design
- Built using CSS Grid and Flexbox
- Optimized for desktop and mobile devices

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API

## Local Storage Usage

The application stores:

1. Theme preference
2. Task data

This ensures that tasks and theme settings remain available after page refresh.
